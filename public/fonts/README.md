# Fontes Gilroy

Esta pasta deve conter os arquivos de fonte Gilroy nos seguintes formatos e pesos:

## Arquivos necessários:

### Light (300)
- Gilroy-Light.woff2
- Gilroy-Light.woff

### Regular (400)
- Gilroy-Regular.woff2
- Gilroy-Regular.woff

### Medium (500)
- Gilroy-Medium.woff2
- Gilroy-Medium.woff

### SemiBold (600)
- Gilroy-SemiBold.woff2
- Gilroy-SemiBold.woff

### Bold (700)
- Gilroy-Bold.woff2
- Gilroy-Bold.woff

### ExtraBold (800)
- Gilroy-ExtraBold.woff2
- Gilroy-ExtraBold.woff

### Black (900)
- Gilroy-Black.woff2
- Gilroy-Black.woff

## Uso:

A fonte já está configurada no Tailwind CSS e pode ser usada com:
- `font-sans` (padrão)
- `font-gilroy`

## Font weights disponíveis:
- `font-light` (300)
- `font-normal` (400)
- `font-medium` (500)
- `font-semibold` (600)
- `font-bold` (700)
- `font-extrabold` (800)
- `font-black` (900)

A fonte será aplicada automaticamente a toda a aplicação como fonte padrão.
